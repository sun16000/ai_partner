# AI 智能伴侣项目

基于 Streamlit 构建的 AI 对话应用，提供会话式 AI 聊天体验，支持多会话管理与历史记录持久化。

---

## 技术栈
- Python
- Streamlit
- JSON 文件存储

---

## 核心功能实现
1.  基础聊天界面布局与消息展示
2.  会话状态管理与上下文记忆
3.  流式输出实现打字机效果
4.  侧边栏会话列表管理
5.  会话的新建、保存、加载与删除
6.  JSON 格式会话数据持久化存储

---

## 核心知识点
- Streamlit 基础组件与会话状态管理
- 文件 IO 操作与资源释放
- JSON 数据序列化与反序列化
- 流式响应处理
- 会话管理模块设计与实现

---

## 当前不足
- 仅支持本地文件存储，无数据库支持
- 未实现用户认证与多用户隔离
- 缺少错误处理与异常捕获优化
